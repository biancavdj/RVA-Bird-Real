🐦 Como usar o AR Bird

1 Coloque o arquivo bird.glb na mesma pasta deste HTML
2 Abra via servidor local:
python -m http.server 8080
ou use o Live Server do VS Code
3 No celular, use um link com HTTPS, como Netlify
4 Aponte para o marcador Hiro e veja o pássaro aparecer ✨
